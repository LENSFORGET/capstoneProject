# Defense Talking Points (Academic RAG + Agent Evaluation)

- The evaluation is not a single score, but a three-level breakdown: RAG retrieval, RAG generation, and Agent trajectory.
- Metrics use common academic sets: Recall/Precision/nDCG/MRR + Faithfulness/Relevancy/Hallucination + TaskSuccess/ToolAccuracy.
- Results report means, 95% CIs, and significance tests, not just average scores.
- Fixed data splits, random seeds, and model/script versions ensure reproducibility.
- Pilot results are only for pipeline validation; thesis main conclusions must come from formal test set runs.
